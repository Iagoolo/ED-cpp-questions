####################################################
### Como compilar e executar esse projeto - Jeito 1
####################################################
Para compilar este projeto: se tive o software 
"make" instalado na sua maquina, va para 
o diretorio deste projeto e digite no 
terminal o comando abaixo:

make

----------------------------------------------

Para executar, digite no terminal do linux:

./main 


Ou no prompt de comando do Windows:

.\main.exe
